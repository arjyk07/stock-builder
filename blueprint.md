
# Blueprint: 대한광통신 투자 분석 사이트

## 개요

대한광통신 주식 투자자들을 위한 차트, 기업 정보, 투자 분석 정보를 제공하는 웹 애플리케이션입니다.

## 프로젝트 구조 및 디자인

- **`index.html`**: 메인 페이지. 헤더, 주요 콘텐츠 영역, 푸터로 구성됩니다.
- **`style.css`**: 모던하고 깔끔한 디자인을 적용합니다. 다크 테마를 기반으로 카드 레이아웃을 사용합니다.
- **`main.js`**: Web Components를 사용하여 각 섹션(차트, 기업 정보, 투자 분석)을 모듈화하고 동적인 기능을 추가합니다.

## 구현된 기능 및 스타일

*   **초기 설정**
    *   다크 테마 기반의 전체적인 스타일링
    *   카드 UI 컴포넌트 레이아웃
    *   헤더, 메인, 푸터의 구조적인 레이아웃

## 현재 작업 계획

1.  **`index.html` 구조 변경**:
    *   페이지 타이틀을 "대한광통신 투자 분석"으로 변경합니다.
    *   헤더에 사이트 제목을 추가합니다.
    *   메인 영역에 "차트 분석", "기업 정보", "투자자 분석" 섹션을 포함하는 3개의 카드를 추가합니다.
    *   푸터에 저작권 정보를 추가합니다.
2.  **`style.css` 스타일링**:
    *   전체 페이지에 다크 테마 (어두운 배경, 밝은 텍스트)를 적용합니다.
    *   카드 형태의 UI를 위한 스타일을 추가합니다.
    *   모바일 반응형 레이아웃을 위한 미디어 쿼리를 추가합니다.
3.  **`main.js` 기능 추가**:
    *   각 섹션을 위한 간단한 Web Component를 생성하여 `index.html`에 적용합니다.
